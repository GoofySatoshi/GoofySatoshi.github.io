# Python入门指南

Python是一门优雅、简单且功能强大的编程语言。本文将带你了解Python的基础知识。

## 为什么选择Python？

- 简单易学
- 可读性强
- 广泛应用
- 丰富的生态系统

## 开发环境搭建

1. 下载Python
2. 安装IDE
3. 配置环境变量

## 基础语法

```python
# 这是一个简单的Python程序
print("Hello, World!")

# 变量定义
name = "Python"
version = 3.9

# 条件语句
if version >= 3.6:
    print(f"You are using Python {version}")
```
